# Get Started with Aztec.js

Welcome to level 1 of the Aztec mission by Level Up!

Aztec.js is a library that provides APIs for managing accounts and interacting with contracts on the Aztec network. In this guide, you'll learn how to deploy contracts, send transactions, and call view functions.

## High Level Structure

1. **GitHub Repository:** Begin by exploring the GitHub repository and README.
2. **Guide:** Proceed to the guide where you will find instructions on deploying contracts, sending transactions, and calling view functions using Aztec.js.

## Mission

1. Install the `aztec.js` package.
2. Build a simple JavaScript code to deploy a contract, send a transaction, and call a view function.

Follow the steps outlined in the guide to complete your mission and level up your skills with Aztec.js!
