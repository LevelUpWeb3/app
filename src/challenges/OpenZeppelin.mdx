# Get Started with OpenZeppelin Contracts Wizard

Welcome to level 1 of the OpenZeppelin mission by Level Up!

OpenZeppelin offers a suite of tools and libraries designed to help developers create, deploy, and manage smart contracts in a secure and efficient manner. In this mission, you'll learn how to use the OpenZeppelin Contracts Wizard to create custom ERC smart contracts for Ethereum.

## High Level Structure

1. **GitHub Repository:** Begin by exploring the GitHub repository and README.
2. **Guide:** Proceed to the guide where you will find instructions on using the OpenZeppelin Contracts Wizard to create an ERC-20 token.

## Mission

1. Visit the OpenZeppelin documentation to understand the services and tools provided.
2. Follow the guide to learn how to access the OpenZeppelin Contracts Wizard, create an ERC-20 token, customize its features, set parameters, and generate Solidity code.
3. Set up your development environment with Hardhat or Truffle and install the OpenZeppelin Contracts library.
4. Integrate the generated Solidity code into your project and deploy your ERC-20 token to a test network.

Complete the mission to level up your skills with OpenZeppelin and start building secure and efficient smart contracts for your blockchain projects!
